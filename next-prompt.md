The Main Mirror Interface acting as the central hub for the MirrorMind AI emotional companion.

**DESIGN SYSTEM (REQUIRED):**
**Vibe**: Futuristic AI OS Companion (MirrorMind). Extremely premium, dark mode only.
**Styling**: Use Tailwind CSS. Backgrounds must be very dark slate/obsidian (`bg-slate-950`). Use intense glassmorphism (`backdrop-blur-xl bg-white/5 border border-white/10`) for all cards and panels.
**Typography**: San-serif (Inter/Outfit). White or light gray text.
**Accents**: Use subtle neon gradients (cyan/purple/amber) for active states, borders, and charts. Elements should feel alive but ambient and supportive.
**Spacing**: Generous padding, clean un-cluttered layouts. Feel like a high-end dashboard, not a dense app.

**Page Structure:**
1. **Header/Status Bar**: Minimal layout showing system time, small AI Companion Notification (e.g., "MirrorMind is observing. You seem focused."), and user profile/settings icon.
2. **Main Camera View Center**: A large, beautifully rounded container representing the live camera feed (use a high-quality placeholder image of a person at a desk). Add futuristic scanning UI overlays (brackets around the face, subtle gridlines).
3. **Emotion Detection Panel (Left or Overlaid)**: Displays the detected mood (e.g., "Focused", "Calm") with a confidence percentage, pulse animation, and a gradient line chart showing mood over the last hour.
4. **Suggestion Engine Panel (Right or Bottom)**: Dynamic glass cards with actionable suggestions. Examples:
   - "You've been focused for 2 hours. Try a 5-minute breathing exercise." (Play button)
   - "Playing deep focus ambient beats." (Mini audio player)
   - "Hydration reminder"
5. **Floating Actions**: Quick toggles for Privacy Mode (disable camera), Fullscreen Zen Mode, and AI Voice Chat.
